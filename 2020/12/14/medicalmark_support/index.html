MedicalMark Support

Anatomical structure point learning tools to establish their own learning library.
Import any picture and add dimension points. Learn while tagging, and export the generated pictures and data at the same time.

Features

  1. Manage as a file and add any picture two, Click on the picture to add notes, English, description, etc.
  2. Configure adjustable display styles such as lines, text, spacing, etc.
  3. Folder directly exports the resulting annotation picture.
  4. Export PDF, SQL data, TEXT data generated by annotations.
